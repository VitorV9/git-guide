# 📚 Guia de Versionamento: Git e Commits Convencionais

Este repositório é um guia prático para consulta rápida de comandos do Git e regras de padronização de mensagens de commit. O objetivo é facilitar o fluxo de trabalho e garantir um histórico de projeto limpo e organizado.

## 🗂️ Sumário

Navegue pelos guias abaixo de acordo com a sua necessidade:

* [📖 Comandos do Git](./comandos-git.md)
  *Uma lista de consulta rápida (cheat sheet) com os comandos essenciais para o dia a dia, desde a clonagem de repositórios até a resolução de conflitos.*

* [🤝 Commits Convencionais](./CONTRIBUTING.md)
  *Um guia explicando a estrutura do Conventional Commits (feat, fix, docs, chore, etc.) e as melhores práticas para padronizar as mensagens do seu código.*

## 🎯 Motivação

Criado para centralizar o conhecimento sobre versionamento de código, servindo como uma documentação de apoio rápido para o desenvolvimento de software, projetos acadêmicos e colaboração em equipe.
